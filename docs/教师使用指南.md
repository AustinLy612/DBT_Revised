# DBT 技能教学平台 — 教师/报告查看者使用指南

## 一、你的角色

作为**报告查看者（report_viewer）**，你可以：

- 查看所有学生的个体可视化报告
- 导出学生报告为 PDF 文件
- 导出学生原始数据（JSON / CSV 格式）
- **不可以**：修改数据、访问后台管理

如需更高级别的权限（如数据导出、用户管理），请联系管理员将你的角色升级为 **admin**。

---

## 二、登录

### 2.1 访问地址

在浏览器地址栏输入：

```
https://<服务器地址>:10443
```

### 2.2 登录

1. 使用管理员为你创建的账号和密码登录
2. 登录后进入首页

> 报告查看者账号由管理员在后台创建，不能通过公开注册页面自行注册。

---

## 三、查看学生报告

### 3.1 进入报告仪表盘

登录后，在浏览器地址栏输入（或由管理员告知链接）：

```
https://<服务器地址>:10443/reports/
```

你将看到一个**学生列表页面**，列出系统中所有学生。

### 3.2 报告页面概览

点击任意学生进入个体报告页面。报告包含 **7 个区块**：

---

#### 区块 1：基础信息

展示学生的：

| 信息 | 说明 |
|------|------|
| 用户名 | 学生账号名 |
| 注册时间 | 账号创建时间 |
| 性别 | 学生问卷填写内容 |
| 年龄 | 学生问卷填写内容 |
| 年级 | 初一至高三 |
| 兴趣爱好 | 多选标签 + 自由填写的其他爱好 |
| 困扰/担忧 | 多选标签 + 自由填写的其他困扰 |

---

#### 区块 2：报告摘要

AI 自动生成的自然语言总结，涵盖学生在以下方面的整体表现：

- 教学参与度
- 测试通过率
- 情绪变化趋势
- 技能掌握情况
- 成就解锁
- 风险事件（如存在）

**阅读建议**：摘要提供了对学生学习状态的快速概览，适合在辅导谈话前快速了解情况。

---

#### 区块 3：概览卡片

以数字卡片形式展示核心指标：

| 指标 | 含义 |
|------|------|
| 完成教学次数 | 该学生完成的 AI 教学会话总数 |
| 测试次数 | 该学生参与的总测试次数 |
| 成就解锁数 | 该学生已获得的成就数量 |
| 总正确率 | 所有测试的综合正确率（百分比） |

---

#### 区块 4：情绪变化

- **SVG 折线图**：展示学生近期情绪值（1-5 分）的变化趋势
- **情绪历史表格**：列出每条心情记录的时间、场景（教学前/教学后/测试后/手动）、心情值和备注

**阅读建议**：关注情绪的整体走势。持续低分（1-2 分）或突然大幅下降值得关注。

---

#### 区块 5：技能学习次数

**CSS 条形图**展示各项 DBT 技能的学习次数，可以直观看出：
- 学生最常练习哪些技能
- 哪些技能尚未接触

---

#### 区块 6：测试记录

表格展示该学生每次测试的详细信息：

| 列 | 说明 |
|------|------|
| 时间 | 测试完成时间 |
| 教学技能 | 该次教学所学的 DBT 技能 |
| 成绩 | 正确数/总题数（如 4/5） |
| 结果 | 通过（≥4/5）或未通过或重测中 |

通过测试记录可以了解学生对各技能的掌握程度。

---

#### 区块 7：成就解锁

以卡片形式展示学生已解锁的成就：
- 成就图标
- 成就名称
- 成就描述
- 解锁时间

---

## 四、导出 PDF 报告

在任意学生的个体报告页面，点击 **"导出 PDF"** 按钮，系统会生成一份 A4 格式的 PDF 文件自动下载。

**PDF 报告包含与网页版相同的内容**（图表和表格均会渲染），适合打印或存档。注意：PDF 中不显示心情 emoji 表情和成就图标（以文字替代），保证中文排版正常。

---

## 五、导出学生数据

在报告仪表盘页面（`/reports/`），除了查看报告外，还可以导出学生的原始数据。

### 5.1 导出入口

在报告仪表盘页面，每个学生右侧有**数据导出**列，包含两个链接：

| 按钮 | 格式 | 用途 |
|------|------|------|
| **JSON** | `.json` 文件 | 结构化数据，适合程序处理或导入其他系统 |
| **CSV** | `.csv` 文件 | 表格数据，适合 Excel / WPS 打开查看 |

页面顶部还有两个批量导出按钮：

| 按钮 | 说明 |
|------|------|
| **导出全部 (JSON)** | 将所有学生的数据打包为一个 JSON 文件 |
| **导出全部 (CSV)** | 将所有学生的数据合并为一个 CSV 文件 |

> 注意：原来的独立导出页面（`/export/`）已合并到报告仪表盘，访问 `/export/` 会自动跳转到 `/reports/`。

### 5.2 JSON 导出包含的数据
- 基本信息（用户名、注册时间、性别、年龄、年级、爱好、困扰）
- 教学记录（所有教学会话、对话消息）
- 测试记录（题目、选项、答案、正误、解析）
- 心情记录（时间、场景、分值、备注）
- 成就解锁记录
- 风险事件（如有）

**CSV 导出**以表格形式呈现相同数据，每个数据表为一个区块，适合直接在 Excel 中查看和筛选。

---

## 六、常见使用场景

### 场景 1：辅导谈话前了解学生状态

1. 打开报告仪表盘 → 点击学生姓名
2. 先看**区块 2（报告摘要）**了解整体情况
3. 再看**区块 4（情绪变化）**关注情绪趋势
4. 最后看**区块 6（测试记录）**了解技能掌握情况

### 场景 2：跟踪学生进步

1. 打开学生报告页面
2. 重点查看**区块 3（概览卡片）**和**区块 6（测试记录）**
3. 对比不同时间的报告，观察完成次数、正确率的变化

### 场景 3：批量了解多名学生

1. 打开报告仪表盘
2. 逐个点击学生进入报告页面
3. 对需要重点跟进的学生导出 PDF 存档

### 场景 4：打印报告用于会议/讨论

1. 打开学生报告页面
2. 点击 **"导出 PDF"**
3. 打印下载的 PDF 文件

## 七、常见问题

**Q: 仪表盘上不显示某个学生怎么办？**
A: 报告仪表盘显示所有具有 `student` 角色的用户。如果你看不到某个学生，请确认该学生已注册。如果问题持续，联系管理员。

**Q: PDF 导出后中文乱码怎么办？**
A: 系统已在 Docker 环境中预装 WeasyPrint 所需的字体依赖（libpango、libcairo），正常情况下中文渲染正常。如仍有问题，告知管理员。

**Q: 我能看到学生的原始对话内容吗？**
A: 可以。通过数据导出功能（`/export/`），你可以导出学生的完整数据（JSON 或 CSV 格式），其中包括所有教学对话消息、测试题目和答案、心情记录等原始数据。

**Q: 学生的数据是实时更新的吗？**
A: 是的。每次你打开报告页面或导出数据，系统都会从数据库实时查询最新数据。

**Q: 我可以修改学生的数据吗？**
A: 不可以。报告查看者只有只读权限。如需修改数据，请联系管理员。

**Q: 如何让另一个老师也能查看学生报告和导出数据？**
A: 联系管理员，在后台为该老师创建 `report_viewer` 账号即可。所有 report_viewer 角色用户均可查看全部学生报告和导出数据。\n\n**Q: 我无法访问数据导出页面（/export/）怎么办？**
A: 请确认你的账号角色为 `report_viewer` 或 `admin`。如果角色正确但仍无法访问，联系管理员检查账号设置。
