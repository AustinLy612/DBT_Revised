# DBT 技能教学平台 — 教师/报告查看者使用指南

## 一、你的角色

作为**报告查看者（report_viewer）**，你可以：

- 查看被授权学生的个体可视化报告
- 导出学生报告为 PDF 文件
- **不可以**：查看原始数据、修改数据、导出 JSON/CSV、访问后台管理

如需更高级别的权限（如数据导出、用户管理），请联系管理员将你的角色升级为 **admin**。

---

## 二、登录

### 2.1 访问地址

在浏览器地址栏输入：

```
https://<服务器地址>:10443
```

### 2.2 登录

1. 使用管理员为你创建的账号和密码登录
2. 登录后进入首页

> 报告查看者账号由管理员在后台创建，不能通过公开注册页面自行注册。

---

## 三、查看学生报告

### 3.1 进入报告仪表盘

登录后，在浏览器地址栏输入（或由管理员告知链接）：

```
https://<服务器地址>:10443/reports/
```

你将看到一个**学生列表页面**，列出系统中所有学生。

### 3.2 报告页面概览

点击任意学生进入个体报告页面。报告包含 **7 个区块**：

---

#### 区块 1：基础信息

展示学生的：

| 信息 | 说明 |
|------|------|
| 用户名 | 学生账号名 |
| 注册时间 | 账号创建时间 |
| 性别 | 学生问卷填写内容 |
| 年龄 | 学生问卷填写内容 |
| 年级 | 初一至高三 |
| 兴趣爱好 | 多选标签 + 自由填写的其他爱好 |
| 困扰/担忧 | 多选标签 + 自由填写的其他困扰 |

---

#### 区块 2：报告摘要

AI 自动生成的自然语言总结，涵盖学生在以下方面的整体表现：

- 教学参与度
- 测试通过率
- 情绪变化趋势
- 技能掌握情况
- 成就解锁
- 风险事件（如存在）

**阅读建议**：摘要提供了对学生学习状态的快速概览，适合在辅导谈话前快速了解情况。

---

#### 区块 3：概览卡片

以数字卡片形式展示核心指标：

| 指标 | 含义 |
|------|------|
| 完成教学次数 | 该学生完成的 AI 教学会话总数 |
| 测试次数 | 该学生参与的总测试次数 |
| 成就解锁数 | 该学生已获得的成就数量 |
| 总正确率 | 所有测试的综合正确率（百分比） |

---

#### 区块 4：情绪变化

- **SVG 折线图**：展示学生近期情绪值（1-5 分）的变化趋势
- **情绪历史表格**：列出每条心情记录的时间、场景（教学前/教学后/测试后/手动）、心情值和备注

**阅读建议**：关注情绪的整体走势。持续低分（1-2 分）或突然大幅下降值得关注。

---

#### 区块 5：技能学习次数

**CSS 条形图**展示各项 DBT 技能的学习次数，可以直观看出：
- 学生最常练习哪些技能
- 哪些技能尚未接触

---

#### 区块 6：测试记录

表格展示该学生每次测试的详细信息：

| 列 | 说明 |
|------|------|
| 时间 | 测试完成时间 |
| 教学技能 | 该次教学所学的 DBT 技能 |
| 成绩 | 正确数/总题数（如 4/5） |
| 结果 | 通过（≥4/5）或未通过或重测中 |

通过测试记录可以了解学生对各技能的掌握程度。

---

#### 区块 7：成就解锁

以卡片形式展示学生已解锁的成就：
- 成就图标
- 成就名称
- 成就描述
- 解锁时间

---

## 四、导出 PDF 报告

在任意学生的个体报告页面，点击 **"导出 PDF"** 按钮，系统会生成一份 A4 格式的 PDF 文件自动下载。

**PDF 报告包含与网页版相同的内容**（图表和表格均会渲染），适合打印或存档。

---

## 五、常见使用场景

### 场景 1：辅导谈话前了解学生状态

1. 打开报告仪表盘 → 点击学生姓名
2. 先看**区块 2（报告摘要）**了解整体情况
3. 再看**区块 4（情绪变化）**关注情绪趋势
4. 最后看**区块 6（测试记录）**了解技能掌握情况

### 场景 2：跟踪学生进步

1. 打开学生报告页面
2. 重点查看**区块 3（概览卡片）**和**区块 6（测试记录）**
3. 对比不同时间的报告，观察完成次数、正确率的变化

### 场景 3：批量了解多名学生

1. 打开报告仪表盘
2. 逐个点击学生进入报告页面
3. 对需要重点跟进的学生导出 PDF 存档

### 场景 4：打印报告用于会议/讨论

1. 打开学生报告页面
2. 点击 **"导出 PDF"**
3. 打印下载的 PDF 文件

---

## 六、常见问题

**Q: 仪表盘上不显示某个学生怎么办？**
A: 请确认：
1. 该学生已注册并具有 `student` 角色
2. 管理员已将该学生授权给你（通过 `ReportViewerAssignment`）
3. 授权处于"激活"状态（`is_active=True`）

如果都不奏效，联系管理员检查授权设置。

**Q: PDF 导出后中文乱码怎么办？**
A: 系统已在 Docker 环境中预装 WeasyPrint 所需的字体依赖（libpango、libcairo），正常情况下中文渲染正常。如仍有问题，告知管理员。

**Q: 我能看到学生的原始对话内容吗？**
A: 当前版本的报告页面不包含完整对话记录。报告摘要基于对话内容生成，但原始对话文本不在报告页面中展示。如需查看原始数据，请联系 admin 角色管理员通过数据导出功能获取。

**Q: 学生的数据是实时更新的吗？**
A: 是的。每次你打开报告页面，系统都会从数据库实时查询最新数据。

**Q: 我可以修改学生的数据吗？**
A: 不可以。报告查看者只有只读权限。如需修改数据，请联系管理员。

**Q: 如何让另一个老师也能查看学生报告？**
A: 联系管理员，在后台为该老师创建 `report_viewer` 账号并建立对应的 `ReportViewerAssignment` 授权。
