# DBT 技能教学平台 — 学生使用指南

## 一、快速开始

### 1.1 你需要什么

- **邀请码**：由老师或研究人员提供，一个邀请码只能注册一个账号
- **设备**：电脑（Windows/Chrome）或平板（iPad/Safari）均可
- **网络**：连接到学校或研究环境的网络

### 1.2 访问地址

在浏览器地址栏输入（具体地址由老师告知）：

```
https://<服务器地址>:10443
```

---

## 二、注册与登录

### 2.1 注册账号

1. 打开平台首页，点击 **"注册"** 按钮
2. 填写以下信息：
   - **用户名**：3 个字符以上，不能与他人重复
   - **密码**：8 个字符以上
   - **确认密码**：再次输入相同密码
   - **邀请码**：输入老师给你的邀请码
3. 点击 **"注册"** 提交
4. 注册成功后自动登录，跳转到问卷页面

> **注意**：邀请码只能使用一次。如果提示"邀请码无效或已被使用"，请联系老师获取新的邀请码。

### 2.2 登录

1. 打开平台首页，点击 **"登录"**
2. 输入用户名和密码
3. 点击 **"登录"**

---

## 三、填写个人问卷

首次登录后，系统会要求你完成一份简短问卷，帮助 AI 了解你的基本情况，提供更个性化的教学。

### 3.1 问卷内容

| 项目 | 说明 |
|------|------|
| 性别 | 选择你的性别 |
| 年龄 | 填写实际年龄 |
| 年级 | 从初一到高三中选择 |
| 爱好标签 | 多选你的兴趣爱好（如阅读、篮球、音乐等） |
| 其他爱好 | 可以自由填写标签中没有的爱好 |
| 困扰标签 | 多选你近期可能存在的困扰（如学业压力、人际困扰等） |
| 其他困扰 | 可以自由填写标签中没有的困扰 |

### 3.2 修改问卷

填写并提交后，可以随时回到问卷页面修改你的信息。修改后的信息会在下一次教学中生效。

---

## 四、开始教学

### 4.1 整体流程

```
记录教学前心情 → 和AI聊聊近况 → AI推荐技能 → 确认技能
    → AI对话教学 → 自动结束 → 记录教学后心情 → 进入测试
```

一次完整的教学大约需要 10-20 分钟。

### 4.2 第一步：开始新教学

1. 在首页点击 **"开始教学"**，进入教学首页
2. 在教学首页查看你的问卷信息
3. 点击 **"开始新教学"** 按钮

### 4.3 第二步：记录教学前心情

- 从 5 个表情中选择最能代表你现在心情的：
  - 😫 很差 → 😟 不好 → 😐 一般 → 🙂 不错 → 😄 很好
- 可选的备注文字（如"刚考完试"、"今天比较累"等）
- 点击 **"确认并开始"**

### 4.4 第三步：和 AI 聊聊你的近况

AI 会问你一个关于近期状况的问题，例如：

> "最近一周，有什么事情让你感到开心或者有压力吗？"

- 在文本框中自由分享（至少写一句话），越详细 AI 越能推荐合适的技能
- 点击 **"分享并开始推荐"**
- 你分享的内容仅用于本次教学推荐，不会被其他人看到

### 4.5 第四步：确认技能

AI 会根据你的问卷信息、心情记录和个人分享，推荐一个最适合你的 DBT 技能。

- 页面会显示**推荐的技能名称**、所属模块和**推荐理由**
- 你可以接受推荐，也可以在输入框中输入自己想学的技能
- 点击 **"确认并开始教学"**

**当前可选技能**（部分示例）：
正念呼吸、观察念头、接纳感受、身体扫描、智慧心、情绪调节等

### 4.6 第五步：AI 对话教学

进入教学对话页面，你会看到：

- **左侧**：教学计划（分步骤展示本次教学的内容结构）
- **右侧**：AI 教学对话区域

**如何参与教学对话**：

1. AI 会先发送一条开场消息，介绍本次要学习的技能
2. 在底部输入框输入你的想法、问题或回答
3. 点击 **"发送"** 或按 Enter 键
4. AI 会逐字生成回复（流式输出），你可以边等边读
5. 继续对话，跟着 AI 的引导完成每个教学步骤

**对话过程中的功能**：

| 功能 | 操作 |
|------|------|
| 语音播报 | 点击 AI 消息旁的 🔊 按钮，朗读该条回复 |
| 自动播报 | 打开"自动播报"开关，每条 AI 回复自动播放语音 |
| 语音输入 | 点击 🎤 按钮，说话转文字（需麦克风权限） |
| 教学配图 | 点击左侧"生成教学配图"按钮，AI 生成情景图片 |
| 结束教学 | 点击底部"结束教学"按钮 |
| 终止会话 | 点击底部"终止会话"按钮（一般不需要用） |

### 4.7 教学完成

当 AI 完成所有教学步骤，会发送一条总结消息，教学自动结束。你会看到：

- **教学摘要**：AI 生成的本次教学要点总结
- **完成时间**和所学技能
- **"记录教学后心情"** 提示
- **"开始测试"** 按钮

---

## 五、测试

### 5.1 进入测试

在教学完成页面点击 **"开始测试"**，系统会创建测试。

- 测试题由 AI 根据本次教学内容自动生成
- 生成需要等待约 30-40 秒（会显示加载动画）
- 如果生成失败，页面会显示"题目生成超时"提示，可以点击"重新创建测试"

### 5.2 答题

- 共 **5 道选择题**，每题 4 个选项（A/B/C/D）
- **逐题作答**：提交一道题后立即显示正误判断和详细解析
- 每题答完后自动显示下一题
- 左侧进度条显示已答/未答状态

### 5.3 测试结果

答完 5 题后点击 **"提交测试"**：

| 结果 | 标准 | 后续操作 |
|------|------|---------|
| **通过** | 答对 ≥ 4 题 | 可以记录测试后心情 |
| **未通过** | 答对 < 4 题 | 可以点击"重新测试"（不限次数） |

每次重测 AI 都会生成全新的题目。

### 5.4 查看测试记录

回到教学会话页面，你可以在"教学已完成"卡片下方看到该次教学的所有测试记录：

- **第 N 次测试**：每次测试的编号
- **通过/未通过**标签
- **正确数**：例如 4/5
- **查看详情**：点击进入测试页面回顾所有题目和解析
- **重新测试**：未通过的测试可以直接从这里重测

## 六、心情记录

### 6.1 心情记录时机

| 时机 | 说明 |
|------|------|
| 教学前 | 在开始教学前自动弹出 |
| 教学后 | 教学完成后提示记录 |
| 测试后 | 测试完成后提示记录 |
| 手动记录 | 随时可以记录 |

### 6.2 手动记录

1. 在教学首页点击 **"记录心情"**
2. 选择 1-5 分的心情值
3. 可选填写备注
4. 点击提交

### 6.3 查看心情历史

在教学首页点击 **"心情历史"**，可以看到你所有的心情记录。

---

## 七、成就系统

### 7.1 查看成就

在教学首页或首页点击 **"成就"**，可以看到：

- 已解锁的成就（彩色图标、名称、解锁时间）
- 尚未解锁的成就（灰色图标、解锁条件）

### 7.2 成就类型

成就由系统自动检测并解锁，触发条件包括：

- 完成首次教学
- 通过首次测试
- 连续学习多天
- 累计完成一定次数教学
- 心情记录相关成就

## 八、风险提示

在教学对话或测试答题过程中，如果你的输入包含可能表示危险状态的内容，系统会：

1. **暂停教学/测试**
2. **显示风险提示页面**，包含：
   - 全国统一心理援助热线：**12356**
   - 北京市心理援助热线：**010-82951332**
   - 联系老师的提示
3. 点击 **"返回安全区域"** 回到首页

这是系统为保障你的安全而设置的保护机制。如果你确实需要帮助，请拨打上面的热线电话或联系身边可信任的成年人。

## 九、常见问题

**Q: 教学到一半可以暂停吗？**
A: 可以。点击对话框底部的"结束教学"按钮正常结束，或直接关闭页面。下次回来可以在教学首页的"最近教学记录"中找到未完成的会话。

**Q: 测试没过可以重来吗？**
A: 可以，不限次数。每次重测 AI 都会生成新题目。

**Q: AI 说话声音可以关掉吗？**
A: 可以。关掉对话页面上方的"自动播报"开关即可。

**Q: 我的数据会被别人看到吗？**
A: 你的教学对话和个人分享仅研究用途，不会对其他学生公开。只有你本人和授权的研究人员/老师可以查看。

**Q: 页面加载不出来或报错怎么办？**
A: 刷新页面重试。如果问题持续，告知老师。
